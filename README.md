# StudyNotes Android

اپلیکیشن اندروید یادداشت‌برداری و مرور درسی با رابط RTL فارسی، دسته‌بندی موضوعات، یادداشت Markdown، نکات علمی، لینک داخلی یادداشت/نکته، تصویر محلی، لینک وب، اتصال به فایل PDF و بازکردن صفحه مشخص PDF با PdfRenderer، SQLite محلی و بکاپ/بازیابی JSON.

## ساخت APK ریلیز امضاشده
در GitHub از تب Actions، workflow `Build signed release APK` را اجرا کنید یا یک tag مثل `v1.0.0` push کنید. workflow یک keystore عمومی می‌سازد و APK ریلیز امضاشده را در artifact و در Release قرار می‌دهد.

> این امضا عمومی است و برای انتشار رسمی بهتر است keystore خصوصی خودتان را جایگزین کنید.

## بازنویسی UI
نسخه فعلی با یک Design System داخلی و بدون وابستگی خارجی بازنویسی شده است: کارت‌های نرم، فاصله‌گذاری ثابت، typography خوانا، RTL کامل، حالت تاریک/روشن، صفحه خواندن مرتب، ادیتور تمیز و ابزارهای درج لینک داخل متن.
